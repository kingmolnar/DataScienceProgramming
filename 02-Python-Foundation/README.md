# Session 2 :: Python programming language
This module provides you with a comprehensive introduction to Python programming. It covers essential concepts such as data types, expressions, control flow, and functions. You will learn to write and execute Python programs, gaining proficiency in utilizing Python's core elements. Additionally, the module explores creating custom Python modules with functions and demonstrates how to import them into notebooks and scripts, enabling you to develop a solid foundation for practical coding and modular programming in Python.

## Outcomes
-	You understand the core elements of the Python programming language.
-	You understand the different data types and know how to write expressions to manipulate values of these types.
-	You understand control flow and functions in Python.
-	You can write programs in Python and execute programs from the command line interface.
-	You write custom Python modules with functions and import to notebooks and other scripts.
